# AI-Machine-Learning
A basic Linear Regression model in Machine Learning.
This repository demonstrates the implementation of Linear Regression, one of the most fundamental algorithms in Machine Learning. Linear Regression is used to model the relationship between a dependent variable (target) and one or more independent variables (features). The project includes:

Data preprocessing
Model training
Evaluation metrics
Visualization of results
